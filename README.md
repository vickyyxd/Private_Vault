# 🔐 Private Vault Calculator

A sleek engineering calculator with a hidden private vault for securely storing photos and videos. Features scientific & engineering calculations, memory functions, secret PIN access, media management, panic mode, lightbox viewer, and a futuristic cyber-themed UI built with pure HTML, CSS, and JavaScript.

### ✨ Features

* 🧮 Standard, Scientific & Engineering Calculator Modes
* 🔒 Hidden PIN-Protected Vault
* 📷 Secure Photo Storage
* 🎬 Secure Video Storage
* ⚡ Panic Mode for Instant Vault Hiding
* 💾 Memory Functions (M+, M-, MR, MC)
* 📐 DEG/RAD Angle Modes
* 🌌 Futuristic Cyberpunk UI Design
* 🖼️ Lightbox Media Viewer
* ⌨️ Keyboard Support
* 📱 Responsive Design

Built with HTML
